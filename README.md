## 1. 项目简介

### 1.1 项目背景与核心理念
#### 1.1.1 传统教育模式痛点
- 教师作业批改负担重，平均每天需2-3小时手动批改，影响教学准备和个性化辅导。
- 教育资源分散，利用率低，学生难以获取精准匹配的资料。
- 统一教学进度难以适应个体差异，影响整体教学质量和学习效果。

#### 1.1.2 项目定位
本项目是一款深度融合AI技术的“智慧课程”教学辅助平台，通过AI高效管理教育资源、智能优化教学流程，提升教学质量和学习效果。

#### 1.1.3 核心理念
- **智能化生产**：自动生成各类教学资料，减轻教师压力。
- **结构化管理**：有序组织教学流程，提高规范性和效率。
- **个性化学习**：为学生定制学习路径，实现“因材施教”。

### 1.2 系统角色与总体架构
#### 1.2.1 核心用户
- **管理员**：系统维护、用户与角色管理、权限设置。
- **教师**：课程设计、作业布置、智能批改、学情分析。
- **学生**：个性化学习、智能推荐、互动答疑。

#### 1.2.2 技术架构
- 前后端分离，基于 RuoYi-Vue 3.8.5 框架二次开发。
- 前端负责界面与交互，后端负责业务逻辑与数据存储。
- 架构低耦合、高响应、易扩展。

#### 1.2.3 AI 赋能闭环
- **内容理解**：通义千问模型自动抽取知识点，构建知识图谱。
- **智能评估**：豆包大模型分析学生学习行为和作业，智能评分。
- **个性化推荐**：基于评估结果推荐学习资源，支持“换一批”。
- **智能批改**：AI一键批改作业，教师可人工调整。

## 2. 功能说明

### 2.1 管理员端功能
- **系统管理**：用户/角色增删改查，权限设置。
- **视频学习管理**：视频资源增删改查、上传、检查。
- **课程能力要求管理**：能力要求的增删改查。
- **学习资源管理**：PPT、题库管理，随机组卷。
- **通知公告**：公告发布与管理，数据安全。

### 2.2 教师端功能
- **AI智能批改**：一键批改作业，AI评分与评语，人工调整。
- **智能组卷与题库管理**：手动/随机组卷，参数灵活控制。
- **学习任务发布**：任务发布、与作业/答题模块联动。
- **学情分析**：多维度数据、可视化图表、教学决策支持。

### 2.3 学生端功能
- **知识图谱导航**：宏观脉络、节点跳转、微型图谱、学习路径引导。
- **AI学习助手**：个人能力图谱、个性化资源推荐、AI智能问答。
- **视频学习深度记录**：精确记录观看行为，支持个性化学习建议。

## 3. 部署指南

### 3.1 环境要求
- JDK：≥8.0
- Vue：2.6.14
- Node.js：14.17.6
- Python：3.8.10
- Flask：2.0.1
- Neo4j-community：4.4.12

### 3.2 安装 Neo4j-community
1. 访问 [Neo4j 官网](https://neo4j.com/) 下载 4.4.12 版本。
2. Windows：双击安装包，按向导安装。
3. 启动：进入 `bin` 文件夹，双击 `neo4j.bat`。
4. 浏览器访问 [http://localhost:7474/](http://localhost:7474/)，初始用户名/密码均为 `neo4j`，首次登录需修改密码。

### 3.3 配置 Python 环境及相关库
1. 安装 Python 3.8.10，勾选“Add Python to PATH”。
2. 安装依赖：
   ```bash
   pip install flask==2.0.1
   pip install neo4j==4.4.0
   pip install pandas==1.3.5
   pip install numpy==1.21.6
   pip install requests==2.26.0
   ```

### 3.4 安装 JDK
1. 访问 [Oracle 官网](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html) 下载 JDK 8。
2. 安装并配置 `JAVA_HOME` 环境变量。
3. 验证：命令行输入 `java -version`。

### 3.5 安装 Vue 和 Node.js
1. 访问 [Node.js 官网](https://nodejs.org/) 下载 14.17.6。
2. 安装 Node.js，命令行输入 `node -v` 验证。
3. 安装 Vue 2.6.14：
   ```bash
   npm install vue@2.6.14
   npm install -g @vue/cli@4.5.15
   ```

### 3.6 项目部署步骤
1. 克隆项目代码到本地。
2. **后端部署**：
   - 进入后端目录，执行：
     ```bash
     mvn clean package
     java -jar 生成的jar包名称.jar
     ```
3. **前端部署**：
   - 进入前端目录，执行：
     ```bash
     npm install
     npm run build
     ```
   - 将 `dist` 文件夹内容部署到 Nginx 等 Web 服务器。
4. **Python 服务**：
   - 进入 Python 项目目录，执行：
     ```bash
     python app.py
     ```

## 4. 使用说明

### 4.1 管理员使用说明
- 登录系统，管理用户、角色、视频、课程能力要求、学习资源、公告等。
- 支持增删改查、批量操作、条件筛选。

### 4.2 教师使用说明
- 登录系统，管理作业、组卷、题库、学习任务、学情分析。
- 一键AI批改作业，人工调整分数与评语。
- 组卷支持手动与随机，题库支持多题型。
- 发布学习任务，可关联作业与答题模块。
- 查看多维度学情数据与可视化图表。

### 4.3 学生使用说明
- 登录系统，查看知识图谱、能力图谱、推荐资源。
- 通过知识图谱导航学习，点击节点跳转资源。
- 使用AI助手进行个性化学习、智能问答。
- 视频学习自动记录行为，系统提供学习建议。
